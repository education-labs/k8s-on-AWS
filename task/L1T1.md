5. Cloud9 에서 key 생성
(key는 작업자 노드 인스턴스에 ssh 액세스 시 사용)

```
ssh-keygen
```

> Enter 3번 입력하면 기본설정으로 key가 생성

6. 생성한 key를 aws ec2에 업로드

```
aws ec2 import-key-pair --key-name "ekskey" --public-key-material file://~/.ssh/id_rsa.pub
```
